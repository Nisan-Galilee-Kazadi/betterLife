# BetterLife front

Stack : Vite + React + Tailwind CSS.

## Scripts

- `npm install` : installe les dépendances.
- `npm run dev` : lance le serveur de dev.
- `npm run build` : build production.
- `npm run preview` : prévisualise le build.

## Structure

- `src/App.jsx` : ton point d’entrée pour les sections de la page.
- `src/index.css` : directives Tailwind + styles globaux.
- `public/` : assets statiques.

Tu peux maintenant remplacer le contenu de `App.jsx` par ta vraie page d’accueil.
